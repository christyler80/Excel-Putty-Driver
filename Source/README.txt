To compile PuttyDriver.exe, download Putty version 0.76 source code

Overwrite with the folowing Putty source files with the versions in this folder:

Windows\window.c
cmdline.c
ldisc.c
putty.h
terminal.h

Note: the *.c.76 files are the original Putty files and retained included for reference.

Compile using Visual Studio 2012 or later (see original Putty folder windows\VS2012 for Visual Studio project solution files).
